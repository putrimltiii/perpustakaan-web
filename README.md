# Perpustakaan Web (PHP Native)

Aplikasi **Perpustakaan Web** berbasis **PHP Native + MySQL** yang digunakan untuk mengelola data buku perpustakaan dengan sistem login dan role (Admin & User).  
Aplikasi ini dibuat sebagai tugas/project pembelajaran pengembangan web.

## Deskripsi Program
Perpustakaan Web adalah aplikasi CRUD (Create, Read, Update, Delete) yang memungkinkan:
- Admin mengelola data buku (tambah, edit, hapus)
- User melihat daftar buku
- Pencarian buku
- Pagination data
- Upload cover buku
- Tampilan modern (Lilac / Ungu) & responsif (mobile full)

---

##  Fitur Aplikasi
